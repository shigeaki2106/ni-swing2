## 10. このサマリー以降に新セッションで起きそうな質問への先回り

### Q: GitHub Actions が失敗した
- Actions タブ → 失敗したジョブをクリック → ステップ展開してログ確認
- よくある原因:
  - DISCORD_WEBHOOK_URL 未設定 → Secrets 再確認
  - yfinance タイムアウト → 再実行で直ることが多い
  - パッケージ不足 → `requirements.txt` 確認

### Q: 通過銘柄が毎日ゼロ
- yfinance の財務データ取得失敗率が高いと自然に起きる
- `core_universe.py` のユニバース拡張 or `financial_filter.py` の閾値緩和を検討
- 楽天CSVより精度が落ちることは想定内

### Q: Python キャッシュで挙動がおかしい
- `__pycache__` フォルダを削除してから実行

### Q: バックテストの数字が変わった
- yfinance のデータ更新で変動する可能性あり
- v3 結果(PF 1.38)は 2026-05-15 時点のスナップショット

### Q: スマホで Streamlit Cloud が見えない
- 米すいんぐの場合: ホーム画面追加した URL を確認
- スリープ状態の場合は最初の数十秒待つ(自動起動)
