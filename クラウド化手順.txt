  トラブルシューティング
══════════════════════════════════════════════════════════════════

Q: GitHub Actions が失敗する
A: Actions タブ → 失敗したジョブをクリック → ステップ展開して確認
   よくある原因:
   - DISCORD_WEBHOOK_URL が未設定 → Step 3 を再確認
   - yfinance がタイムアウト → 再実行で直ることが多い
   - 財務データ取得に時間がかかる → timeout-minutes 30 に設定済み

Q: Discord に通知が来ない
A: 1. GitHub Actions の実行ログを確認
   2. Webhook URL が正しいか確認
   3. iPhone の Discord 通知設定を確認

Q: 通過銘柄が毎日ゼロ
A: 楽天 SPF の閾値より、yfinance の財務データが取得しにくい銘柄が
   多いため、通過率は低くなりがちです。
   core_universe.py を編集してユニバースを広げる、または閾値を
   緩めることを検討してください。
