## トラブルシューティングメモ

### OneDrive同期遅延（既知）

- Edit/Write tool → Windows側に正しく書き込まれる（Read toolで確認可能）
- bash mount（`/sessions/.../mnt/すいんぐ/`）→ 同期遅延でnullバイトが末尾に付くことがある
- **対策**: 構文チェック時は `data.rstrip(b'\x00').decode('utf-8')` でnullバイトを除去してからast.parse

```python
data = open('swing_analyzer.py','rb').read().rstrip(b'\x00')
src = data.decode('utf-8')
ast.parse(src)
```

### f-string 内に辞書リテラルを書くとSyntaxError

- `f"... {{'A':'x','B':'y'}.get(v,'z')} ..."` → Python 3.10以前でエラー
- **対策**: ヘルパー関数（例: `grade_cls(g)`）に切り出して f-string の外で評価する

### regex の挙動差異

- 古い `.pyc` キャッシュが原因のケースあり
- trade_review.py は手書きスキャン方式に書き換え済み
